Developer guide:

Refer to the README file in the GitHub repository for a detailed guide. This is my first time working with Ruby (I usually use Docusaurus for similar tasks). With this setup, I do not always have to build the website and deploy it myself. Since GitHub Pages will be taking care of it for me, I can edit anything on the go and push the commits, and everything will be updated automatically within a couple of seconds.

Summarized Instructions

# => Download and install Ruby (with Devkit)

#

# => Verify Installation

~ $ ruby -v

#

# => Install bundler

~ $ gem install bundler

#

# => Backup old Gemfile.lock

~ $ cd rename Gemfile.lock Gemfile.lock.bak

#

# => Install Dependencies

~ $ bundle install --verbose

#

# => Start local server

~ $ bundle exec jekyll serve

# => Now browse to http://localhost:4000
