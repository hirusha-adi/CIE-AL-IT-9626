Summarized Instructions

# => Download and install Ruby (with Devkit)

#

# => Verify Installation

~ $ ruby -v

#

# => Install bundler

~ $ gem install bundler

#

# => Backup old Gemfile.lock

~ $ cd rename Gemfile.lock Gemfile.lock.bak

#

# => Install Dependencies

~ $ bundle install --verbose

#

# => Start local server

~ $ bundle install --verbose

# => Now browse to http://localhost:4000
